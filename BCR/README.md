# Data Analyst Assessment – Real Estate Case Study

This repository contains my solution for a real estate data analyst case study.

## Files
- **BCR - Real Estate Data.csv** → Original dataset (raw).
- **BCR_Flagged.csv** → Cleaned dataset with outlier flags.
- **BCR_RealEstate_Clean.csv** → Clean dataset (outliers removed, 184 rows).
- **Data Analysis & Visualization.ipynb** → Jupyter notebook (data cleaning, descriptive stats, visualization).

## Key Insights
- Average price per m² ~ 2,400 AZN (median ~2,354 AZN).
- District segmentation: Sabail & Nasimi premium, Absheron & Binagadi affordable.
- Area effect negligible (correlation ≈ 0.06). Location is the main driver.
